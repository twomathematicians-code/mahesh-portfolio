# Mahesh Solanki - Portfolio Website

🦞 **Data Scientist | Mathematician | AI/ML Engineer**

A high-tech personal portfolio website showcasing expertise in Business, Finance, Healthcare, and Defense domains.

## 🌟 Features

- ⚡ **High-Tech Design** - Dark theme with neon accents and smooth animations
- 🎨 **Interactive Particles** - Dynamic background particle system
- 📱 **Fully Responsive** - Works on all devices
- ⚛️ **React 18** - Built with modern React and hooks
- 🎭 **Framer Motion** - Smooth animations and transitions
- 🎯 **SEO Optimized** - Meta tags and Open Graph support

## 🚀 Live Demo

Visit: [https://twomathematicians-code.github.io](https://twomathematicians-code.github.io)

## 📦 Tech Stack

- **Frontend**: React 18, TypeScript
- **Animations**: Framer Motion
- **Particles**: tsparticles
- **Icons**: React Icons
- **Styling**: Pure CSS with CSS Variables

## 🛠️ Installation

```bash
npm install
npm start
```

## 🏗️ Build

```bash
npm run build
```

## 🚀 Deploy

```bash
npm run deploy
```
