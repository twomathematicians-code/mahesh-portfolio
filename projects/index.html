BiasCorrection-SEM

R package implementing Bartlett Correction for Likelihood Ratio Test bias reduction in Structural Equation Modelling. First computational framework for SEM bias reduction with applications in psychology and healthcare.

Key Features

  • Bias-corrected likelihood ratio tests
  • Integration with lavaan SEM package
  • Accurate model selection in finite samples
  • 23% reduction in Type I error rates

Bias-corrected LRT statistic:

$$\Lambda^* = \Lambda \left(1 + \frac{b}{n}\right)$$
